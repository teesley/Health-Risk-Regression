Using the dataset mentioned on the Data Details page, I used SKLearn's Random Forest Classifier Algorithm along with many other tools to predict maternal health risk with an accuracy of 87%. Using the dataset, a user can enter health related information and get a prediction on their maternal risk level. It is important to note that this model should not be used as a primary source of determining maternal health risk as it does not have a 100% accuracy but can be a helpful suplementary resource.
